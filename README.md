# AI 识字宝 📚

一款面向学龄前儿童（3-5岁）的识字教育 PWA 应用，结合游戏闯关、互动绘本和知识卡片多种形式，利用 AI 语音识别提供个性化学习体验。

## ✨ 特性

### 🎮 游戏闯关模式
- 听音辨字、看图识字、朗读测试
- 星级评定系统
- 积分和奖励机制

### 📇 识字卡片模式
- 翻转动画学习
- 语音朗读功能
- 分级浏览（Level 1-3）

### 📊 学习报告
- 学习统计和进度追踪
- 汉字本展示
- 关卡星级进度

### 🎯 适合儿童
- 大按钮设计
- 鲜艳色彩
- 生动动画
- 语音引导

### 📱 PWA 应用
- 可安装到桌面
- 支持离线使用

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

访问 http://localhost:5173

## 🎨 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 新一代前端构建工具
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Tailwind CSS** - CSS 框架
- **Vite PWA** - PWA 插件
- **Web Speech API** - 语音识别

## 📚 汉字数据库

当前包含 75+ 常用汉字，分为 3 个级别：

### Level 1 (3岁)
基础汉字：人、大、小、上、下、天、地、日、月、水、火等 (25个)

### Level 2 (4岁)
数字和家人：一到十、爸爸妈妈、兄弟姐妹等 (26个)

### Level 3 (5岁)
动物和物品：花、鸟、鱼、车、飞机等 (26个)
